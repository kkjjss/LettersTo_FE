## Stack

- react
- react-native
- react-navigation
- zustand
- react-query

## Dependency

- Node.js: v16.13.1
- Watchman: 2-22.09.19.00
- Ruby: 2.7.5
- Xcode: 14.0(14A309)
- cocoapods: 1.11.3
- ReactNative: 0.70

## Installation

```
  $ npm install
  
  // ios
  $ cd ios
  $ pod install
  
  // metro 서버 실행
  $ npm start
  
  // 시뮬레이터 또는 핸드폰에서 앱 실행
  $ npm run android
  $ npm run ios
```
